The Challenge
ปัจจุบันการรับเรื่องร้องเรียนในองค์กรส่วนท้องถิ่น ยังขาดระบบที่มีประสิทธิภาพ ทำให้เกิดความล่าช้าในการดำเนินการ ข้อมูลกระจัดกระจาย และขาดการติดตามสถานะ โปรเจกต์นี้พัฒนาระบบรับแจ้งเรื่องร้องเรียนผ่าน LINE Official Account โดยนำ AI วิเคราะห์และจัดการเรื่องร้องเรียนโดยอัตโนมัติ เพื่อยกระดับการบริการและความพึงพอใจของผู้ใช้งาน
🔧
แนวทางการแก้ไขและฟังก์ชันหลัก
Solution & Key Functions
📲
รับแจ้งเรื่องร้องเรียนผ่าน LINE Official Account และ Web Portal โดยมีหมวดหมู่ 7 ประเภท (ถนน, น้ำ, ไฟฟ้า, ขยะ, สัตว์, ต้นไม้, ภัยพิบัติ) พร้อมระบบ GPS
🤖
ใช้ AI (Anthropic Claude) วิเคราะห์และจัดหมวดหมู่เรื่องร้องเรียนอัตโนมัติ พร้อมคำนวณ Priority Score
⚡
Real-time updates ด้วย Socket.io — Admin Dashboard อัปเดตทันทีเมื่อมีเรื่องร้องเรียนใหม่
📊
ระบบ SLA ติดตามกำหนดเวลา — urgent 8 ชม., medium 48 ชม., normal 72 ชม. พร้อมแจ้งเตือนเมื่อเกินกำหนด
🗺️
Heatmap แสดงจุดร้องเรียนบนแผนที่ (Leaflet.js) และระบบ Admin Dashboard สำหรับผู้บริหาร
⭐
ประชาชนประเมินผลงานผ่าน LINE LIFF Rating (1–5 ดาว) หลังเรื่องร้องเรียนได้รับการแก้ไข
